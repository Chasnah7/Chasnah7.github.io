About Rakko Project

The design philosophy of this website is to be as light and as compatible as possible. Built entirely within HTML 3.2 standards, predominately within Windows 95, on real hardware, by hand. No usage of client-side scripts, trackers, or cookies. Will look best under Internet Explorer 4.0 and newer browsers.
Note that the name and layout are not yet finalized. Rakko is a project encompassing several works I am planning beyond just this website.
